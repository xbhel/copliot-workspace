# Architecture Design

## Goal

Produce a clear, implementation-ready architecture proposal for the target change, including major components, boundaries, interfaces, and trade-offs.

## When to Use

Use this skill when:

- implementation should wait until architecture decisions are clear
- boundaries, interfaces, or responsibilities need to be defined
- multiple design options or trade-offs need structured comparison
